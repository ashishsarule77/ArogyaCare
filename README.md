Hospital Appointment Management System (Django)
A Django-based web application for managing hospital appointments, departments, doctors, and patients.

Project Overview
This project allows users to register, log in, and book medical appointments online. Admin users can manage doctors, departments, and appointments using the Django admin panel.

Tech Stack
Backend: Python, Django
Database: SQLite / MySQL
Frontend: HTML, CSS, Bootstrap, JavaScript
Version Control: Git & GitHub
Key Features
User authentication (login & registration)
Appointment booking system
Department and doctor management
Django admin dashboard
Secure form handling and validation
UI & Backend Disclaimer
Frontend UI adapted from a publicly available template; backend architecture and business logic fully designed and implemented using Django.
